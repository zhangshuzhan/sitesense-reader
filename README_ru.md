## Обзор

RSS Reader — это настольное приложение на Tauri 2 для чтения RSS, Atom и JSON-лент. Оно хранит данные локально в SQLite, снижает стоимость обновления лент с помощью условных запросов и добавляет необязательные ИИ-сценарии для сводок, перевода и оценки статей.

Приложение следует нативной логике macOS: `Command+W` закрывает окно, но оставляет приложение активным в Dock, а `Command+Q` полностью завершает работу приложения.

## Возможности

### Чтение и управление лентами
- Подписка на RSS, Atom и JSON-ленты.
- Импорт и экспорт подписок через OPML.
- Просмотр всех, непрочитанных, отмеченных звездой и избранных статей.
- Организация статей по лентам, тегам и группам.
- Локальный полнотекстовый поиск по статьям.
- Виртуализированный список для больших коллекций статей.

### Производительность и фоновые задачи
- Локальное хранение статей, лент, правил и настроек.
- Использование `ETag` и `Last-Modified`, чтобы не загружать неизменившиеся ленты.
- Обновление лент в Rust с ограниченной параллельностью.
- Легкий фоновый планировщик после закрытия главного окна.
- При отсутствии окна приостанавливаются тяжелые UI- и ИИ-задачи.
- Рендеринг статей, очистка HTML, Markdown-парсинг и подсветка кода загружаются только при необходимости.
- Ограниченный прокси `rss-media://` для медиа, которым нужен кэш или Range-запросы.
- Встроенные видео загружаются только после действия пользователя.

### Необязательные ИИ-инструменты
- Настройка профилей OpenAI или Anthropic-совместимых провайдеров.
- Создание сводок отдельных статей.
- Перевод содержимого статей.
- Создание сводок по нескольким статьям.
- Правила автоматизации и ИИ-оценки для классификации или выделения статей.
- API-ключи хранятся в локальных настройках приложения.

### Настольный опыт
- Нативное поведение меню macOS для закрытия, повторного открытия, скрытия и выхода.
- Сочетания клавиш с переключателем включения в настройках.
- Светлая, темная и системная темы.
- Контекстные меню и пакетные действия со статьями.
- Интерфейс на английском, китайском, русском, испанском, французском и арабском языках.

## Что нового в 0.2.0

- Стандартный жизненный цикл macOS: `Command+W` закрывает окно, `Command+Q` завершает приложение.
- Меньшее потребление ресурсов в скрытом состоянии за счет уничтожения WebView при закрытии окна.
- Фоновое обновление и очистка перенесены в Rust.
- Условная загрузка лент через `ETag` и `Last-Modified`.
- Ленивый рендеринг статей и более легкая загрузка медиа.
- Переключатель сочетаний клавиш в настройках.
- Исправлены восстановление маршрута, навигация из настроек, обновление счетчиков лент и синхронизация статуса прочтения.

## Скачать

Готовые сборки публикуются на странице [GitHub Releases](https://github.com/JinxinWonderWorld/RSS-Reader/releases).

Текущая цель релиза — macOS. Поддержка Windows и Linux остается в конфигурации Tauri, но релизное тестирование сейчас сосредоточено на macOS.

## Разработка

### Требования
- [Node.js](https://nodejs.org/) 18 или новее
- [Rust](https://www.rust-lang.org/tools/install) 1.70 или новее

### Быстрый старт

```bash
git clone https://github.com/JinxinWonderWorld/RSS-Reader.git
cd RSS-Reader
npm install
npm run tauri:dev
```

### Полезные команды

| Команда | Описание |
| --- | --- |
| `npm run dev` | Запустить только Vite-фронтенд |
| `npm run build` | Проверить типы и собрать фронтенд |
| `npm run tauri:dev` | Запустить полное Tauri-приложение в режиме разработки |
| `npm run tauri:build` | Собрать релизный пакет приложения |
| `npm test -- --run` | Запустить фронтенд-тесты |
| `npm run lint` | Запустить ESLint |
| `cargo test --manifest-path src-tauri/Cargo.toml` | Запустить Rust-тесты |

## Архитектура

- `src-tauri/src/app_runtime.rs`: состояние runtime, фоновое планирование и условия очистки.
- `src-tauri/src/window_lifecycle.rs`: закрытие, повторное открытие и восстановление состояния окна macOS.
- `src-tauri/src/feed/`: загрузка лент, условные запросы и парсинг.
- `src-tauri/src/db/`: схема SQLite и доступ к данным.
- `src-tauri/src/media_protocol.rs`: ограниченный медиа-прокси и Range-ответы.
- `src-tauri/src/ai.rs`: ИИ-сводки, перевод, пакетные сводки и обработка очереди.
- `src/services/runtime.ts`: мост фронтенда к Rust runtime-командам.
- `src/stores/`: Zustand-хранилища для лент, настроек, правил, UI-состояния и истории поиска.
- `src/components/`: React-компоненты интерфейса и ленивый рендеринг статей.
